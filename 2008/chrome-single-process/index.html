Chrome 的单进程模式

发表于 2008-09-04 分类于 Gadget
本文字数： 513 阅读时长 ≈ 1 分钟

　　好几天没上网了，这次多发几篇。

　　Chrome的一个很重要的特点是它的一个Tab占一个进程，为了防止崩溃了所有Tab全关，所以多开几个Tab后看着任务管理器真是不爽，至少我觉得不爽。

　　当然，它也有单进程模式（TopLanguage牛人多啊）。使用方法嘛，chrome.exe –single-process 就OK了，再加上从小众那看到的profile本地化的  –user-data-dir 那就无敌了，哈哈。

　　具体使用方法，首先新建一.Bat文件，然后粘贴start “” chrome.exe –single-process –user-data-dir=”%CD%\profile”，一切OK，以后直接使用这个Bat启动Chrome就行了。

　　内存占用还不错，3个Tab，一个Google Reader，两个Googe Groups，37M，同等条件下我加了一版插件的Firefox72M（不过firefox设置了最小化释放内存后就不一样了），虽然没可比性，但是Chrome的内存占用从我看来是不错的。

　　如果你觉得用Bat不爽 ，又难看又没水平。那你可以去找个EXEscript把它编译成EXE，在加上一个图标。恩，那就完美了。下载 :Nami RayFile
