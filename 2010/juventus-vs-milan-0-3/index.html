Juventus VS Milan 0-3

发表于 2010-01-10 分类于 Football
本文字数： 519 阅读时长 ≈ 1 分钟

　　又是一个看球的夜晚，哈哈没白熬夜。不错的过程+不错的运气=3:0,没有比这更令人兴奋的结果了。

　　没想到尤文今天的战斗力如此之差，应该还是费拉拉的原因，整场比赛没见尤文打出什么配合。。这就成就了莱昂纳多在战术方面的“神奇”，

　　一开场米兰更注重于防守，俩边后卫一直没压上,小贝和小罗也经常回撤接应，那会尤文的防守相当硬朗，下脚一直黑到比赛结束，米兰的配合也受了点影响，地面的传接相当难，特别是小罗，被逼的没什么发挥的机会。后来两边卫压上助攻后就好了很多，边路小罗的压力有所改善，米兰也慢慢地有了些配合，再加上尤文攻击乏力，比赛就从此走上了正轨，米兰一直控制到比赛结束。

　　今天全队的发挥都还行，博烈洛在接应方面做得不错，就是在进攻端稍微差了点，总感觉有点犹豫，好几个机会其实应该直接射门的。另外今天米兰的两边表现相当不错，阿巴特和安东尼尼攻守都有出彩的地方，看来圈哥和赞布罗塔算是伤到时候了。不过貌似替补是个问题，这俩位置没啥能打的人了，给小孩一个机会吧。话说今天莱昂纳多才用了两个换人名额，真是节省，因扎吉已经N场没机会了。。

　　意大利的极端球迷真是名不虚传，到比赛结束那段球场上浓烟滚滚+爆炸声隆隆，唉，估计下来尤文又得被罚钱了。可怜的尤文球迷。。。
